Знакомство с комнадой

Впервые придя на занятие по проектной деятельности мы решили разбиться на несколько подкоманд,
которые изучали определенный блок информации, чтобы в будущем использовать ее для развития проекта.
В одном проекте мы были в разных подгруппах: веб-разработка и работа с виртуальной машиной

Определение задач и состава команд

Для команды веб-разработки был выбран вектор напрваления: изучение разметки с основ, чтобы у членов команды было представление
о том как правильно создавать хорший сайт.

Для другой команды был выбран вектор напрваления: создание виртуальной машины на Linux, мы практиковались в работе с консолью, попробовали настроить SSH сервер, подключились к нему с хостовой машины. Эти навыки нужны, чтобы у членов команды было представление о том, что такое виртуальная машина и как работать с серверами

Выбор инструментов разработки

Для команды веб-разработки Состоялось обсуждение испоьлзования шаблонов сайтов для реализации своих собственных страничек,
использования нормализатора для корректного отображения информации, добавления изображений и gif. Каждую неделю мы присылали
нашему куратору выполенныезадания и получали обратную связь для доработки. По итогу первой работы получилось создать
двухстраничный сайт с описанием себя

Для другой команды: мы изучили такие инструменты как vagrant и virtualbox, попрактиковались в установке и настройке различных ОС. Каждую неделю мы присылали
нашему куратору выполенные задания и получали обратную связь для доработки.

Итог работы в группе

Этап первой аттестации для команды веб-разработки был завершен с презентацией группого проекта
подгруппы, где ребята делились на 2-3 человека. Этот проект был напрвален на разработку тематического многостраничного
сайта с представлением предметной области по тематике, которую нам подбирал куратор.

Этап первой аттестации для второй команды был завершен с презентацией группого проекта
подгруппы, в которой мы прописали все выполненные нами задачи.

Итог работы c вариативной частью практики

В рамках вариативной практики были рассмотрены Политики в отношении обработки персональных данных пользователей сайтов других вузов, на основе их структуры и содержания был создан собственный уникальный документ Политики в отношении обработки персональных данных пользователей сайта Московского Политехнического университета. С итоговым файлом вы можете ознакомиться в нашем репозитории выполнения практике в папке docx.
